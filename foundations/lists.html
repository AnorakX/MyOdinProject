How to create lists in HTML

Unordered lists, or "ul"

  • This is a list item, created using the "li" element.
  • "li" is used in both unordered and ordered lists.
  • Since it is unordered, each item begins with a bullet point.

Ordered lists, or "ol"

  1. "li" is used in both unordered and ordered lists.
  2. Since it is ordered, each item begins with a number.

Description lists, or "dl"

"dt" is used for the main point...
followed by dd for its descriptions.

You could do some fun things with ordered lists, like start them at a certain number with the "start" attribute:

  1. Started from the
  2. bottom now we're
  3. here?

Change the value of list items with the "value" attribute:

  1. We've got a long way to go
  2. Maybe not?
  3. Keep climbing...
  4. Uh-oh.
  5. Oof!
  6. One step at a time.

Or even reverse them with the "reversed" attribute:

  1. But now we're going
  2. back to the
  3. bottom
  4. :(

You could nest lists within each other:

  1. Eat breakfast
  2. Go to the grocery and buy:
    • Milk
    • Bread
    • Cheese
  3. Make dinner

    4. (Works for any combination of ul and ol.)
